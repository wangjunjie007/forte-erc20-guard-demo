# Security Policy

## Supported versions

This repository is currently maintained as a demo / showcase project. The latest `main` branch is the supported version.

## Reporting a vulnerability

Please do not open a public issue for a security-sensitive report.

Use one of the following:
- GitHub Security Advisories for the repository, if enabled
- a private disclosure channel controlled by the repository owner

## Notes

- This repo is designed for local demonstration and educational use
- It should not be treated as production-ready token compliance infrastructure without a full security review
- Upstream dependencies may have their own advisory lifecycle
